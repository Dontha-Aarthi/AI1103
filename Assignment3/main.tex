Let $X$ be the Poisson random variable with parameter $\lambda$ = 1. Then, the probability 
$\pr{2 \leq X \leq 4}$ equals ............
 
\section{solution}
Let 
\begin{align}
	X\in \{0,1,2,3,4,5...\}
\end{align}
We know that, for a poisson random variable $X$ with a given parameter $\lambda$, probability of $X=k$ is:
\begin{align}
	\pr{X=k}=\left(\frac{\lambda^k e^{-\lambda}}{k!}\right)	
\end{align}
We have to find the value of $\pr{2 \leq X \leq 4}$ and given that $\lambda$=1.
So,
\begin{multline}
	\pr{2 \leq X \leq 4}= \\ 
	\pr{X=2}+\pr{X=3}+\pr{X=4}
\end{multline}
\begin{equation}
	\begin{split}
	\pr{2 \leq X \leq 4}
	&=\left(\frac{\lambda^2 e^{-\lambda}}{2!}\right)+\left(\frac{\lambda^3 e^{-\lambda}}{3!}\right)+\left(\frac{\lambda^4 e^{-\lambda}}{4!}\right)\\
	& = \left(\frac{e^{-1}}{2!}\right)+\left(\frac{e^{-1}}{3!}\right)+\left(\frac{e^{-1}}{4!}\right)\\
	&=\frac{e^{-1}}{2}+\frac{e^{-1}}{6}+\frac{e^{-1}}{24}\\
	&=\frac{17}{24e}
	\end{split}
\end{equation}
